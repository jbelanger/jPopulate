What is jPopulate?

The jPopulate is a jQuery plugin used to auto populate any html element from a javascript object. The object can contains simple literal properties, other objects, arrays and even arrays of objects. The following structure is supported:

JSON: 

    {
        "Literal" : "I am a literal",
        "Object" : {"Property1" : "prop", "Property2" : "prop2"},
        "Array": [ "item1", "item2", "item3" ],
        "ArrayOfObjects" : [ {"SomeProperty" : "value1", "SomeOtherProperty" : "prop"}, {"SomeProperty" : "value2", "SomeOtherProperty" : "prop"} ]
    }

It uses caching internally so it can work with very large forms.

Note: Requires jQuery 1.5 or higher.

How it works?

jPopulate uses special attributes set on html elements to identify which properties in the object it needs to bind.

Example using the json string above:

Literal binding: 
    <input type="text" data-field="Literal" />
Object binding: 
    <input type="text" data-field="Object.Property1" />
Array binding: 
    <div data-field="Array/" >
        <input type="checkbox" value="item1" />
        <input type="checkbox" value="item2" />
        <input type="checkbox" value="item3" />
    </div>
Array of objects binding: 
    <table data-field="ArrayOfObjects/SomeProperty" >
        ...
        <tr data-dataitemid="value1">
        <input type="checkbox" value="value1" data-bind="SomeProperty"/>
        <input type="text" data-bind="SomeOtherProperty" />
        </tr>
        <tr data-dataitemid="value2">
        <input type="checkbox" value="value2" data-bind="SomeProperty"/>
        <input type="text" data-bind="SomeOtherProperty" />
        </tr>
        ...
    </table>

About the syntax

The "." is used for accessing some properties of an object.

Let's say you have this object 

            { "myObj" : { "Id" : "1", "Name" : "Joe Smith" }}
The Id property can be bound using 
            <input type="text" data-field="myObj.Id" />

The "/" is used to mark the field as being an array. It is necessary to be able to convert the form back to an object.

When working with array of objects, a unique property must also be added after the "/" to tell jPopulate that it needs to bind each dataitemid to this unique property.
